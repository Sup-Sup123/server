Introducing FastChat

FastChat is a multiplatform chat client that focuses on group communication. We were frusterated at the other cross platform solutions for our simple needs, so we made a better one.

We focused on speed and simplicity, working hard to make a cross platform solution that would beat texting.

Speed

From the very beginning, FastChat had to have instant communication, no matter where we were. From the ground up we've built this in.

Simplicity

We're starting with getting the building blocks right. Everything is right where you think it is, with a silky smooth interface.

Feedback

We value your feedback. We want you to let us know what you love, and what you want to see more of. In fact, we're so serious about getting great feedback, you'll have a group with the developers right from the start.

The Server

Socket.io has been used along with Express to make a super simple API to work with. This could allow for many more clients in the future. Interested? Contact us.

The Clients

Each client is built in their native language, focusing on design and speed, making the experience as great as possible for you.

The Team
